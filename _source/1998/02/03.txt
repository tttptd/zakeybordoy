Нажми на кнопку

Специальные символы при работе в MS Word 95/97

Длинное тире = Alt + Ctrl + минус на цифровой клавиатуре
Короткое тире = Ctrl + минус на цифровой клавиатуре
Мягкий перенос = Ctlr + -
Неразрывный пробел = Ctrl + Shift + пробел
Значок копирайта = Alt + Ctrl + c
Охраняемый знак = Alt + Ctrl + r
Товарный знак = Alt + Ctrl + t
Многоточие = Alt + Ctrl + точка
Открывающий апостроф = Alt + ё, ё
Закрывающий апостроф = Alt + э, э
Открывающая кавычка = Ctrl + ё, Shift + э
Закрывающая кавычка = Ctrl + э, Shift + э
